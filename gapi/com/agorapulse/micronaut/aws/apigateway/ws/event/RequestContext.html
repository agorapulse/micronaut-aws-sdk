Package: com.agorapulse.micronaut.aws.apigateway.ws.event

[Java] Class RequestContext

    • com.agorapulse.micronaut.aws.apigateway.ws.event.RequestContext

    • Request context sent with WebSocket event. The most important part is the connectionId which hold the reference which can be used to send message back to connected client and routeKey which points to the matched route from the API Gateway.
